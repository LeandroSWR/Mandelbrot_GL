# Mandelbrot_GL
A C# application to generate the Mandelbrot fractal with interactive controls using OpenGL.

### Controls

- Middle Mouse Click: Move Around
- Scroll Wheel: Zoom In/Out
- Left Shift + Scroll: Change number of iterations
